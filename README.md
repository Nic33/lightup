# lightup

## Version Text

Pour tester le projet sur le terminal, placez-vous à la racine du projet, puis tapez les commandes suivantes :

```
mkdir build
cd build
cmake .. 
make
```

Placez-vous dans le fichier build, puis tapez la commande suivante : 

```./game_text```

## Version SDL2

Placez-vous dans le fichier build, puis tapez la commande suivante : 

```./game_sdl```
